# wiki-List_of_alternative_country_names-extractor

Extract contries alternative names info from [this wiki page](https://en.wikipedia.org/wiki/List_of_alternative_country_names)

## dependency

- python3
- see requirements.txt

## usage

command:
```
python run.py | tee /tmp/output.txt
```

output.txt:
```
Abkhazia,Republic of Abkhazia,Aphsny Axwynthkharra,Respublika Abkhaziya,Autonomous Republic of Abkhazia
Afghanistan,Islamic Republic of Afghanistan,Da Afġānistān Islāmī Jumhoryat,Jomhūrīyyeh Eslāmīyyeh Afġānestān
Albania,Republic of Albania,Republika e Shqipërisë,Arnavutluk,Arbanon
...

Zambia,Republic of Zambia,Northern Rhodesia
Zimbabwe,Republic of Rhodesia,Republic of Zimbabwe,Rhodesia,Southern Rhodesia

```
